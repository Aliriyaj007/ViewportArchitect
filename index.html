User Guide

Overview: ViewportArchitect allows you to test responsive web designs across multiple screen sizes simultaneously without resizing your browser.

  • URL: Enter any web URL or local file path (e.g., example.com or file:///C:/Users/...). For local files, ensure you have permission to access them.
  • Paste HTML: Click "Paste HTML" to directly paste a complete HTML file content. Works with single-file web apps!
  • Local Files: Use file:// protocol (e.g., file:///C:/folder/file.html) or simply paste a full path like C:\Users\folder\index.html
  • Zoom: Use the global slider to scale viewports down to fit your monitor.
  • Custom Devices: Add specific dimensions via the sidebar.
  • Themes: Choose from 8 premium themes to suit your environment.
  • Mobile: Use the ☰ menu button to access sidebar on mobile devices.

Note: Local file access may be restricted by your browser's security settings. For best results with local files, open this app via file:// protocol (double-click the HTML file).
